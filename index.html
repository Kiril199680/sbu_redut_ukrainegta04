Загальні положення

Стаття 82 // Статус
Департамент безпеки «Редут» є спеціалізованим оперативним підрозділом СБУ, який здійснює виявлення, оперативну розробку, документування, припинення та нейтралізацію діяльності організованих злочинних угруповань.
Стаття 82.2 // Юрисдикція
ДБ «Редут» здійснює діяльність на всій території України незалежно від територіальної підпорядкованості інших правоохоронних органів.

Основні завдання та функції

Стаття 87 // Пріоритети
Встановлення структури, ієрархії, ролей учасників та фінансового забезпечення злочинних організацій; виявлення місць незаконного зберігання зброї та оперативний супровід спецоперацій.
Стаття 90 // Інструменти операцій
Проведення оперативно-розшукових, агентурних, інформаційно-аналітичних та документувальних заходів, а також забезпечення стабільного функціонування агентурної мережі.
